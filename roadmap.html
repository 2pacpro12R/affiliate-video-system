Phase 04 — En cours
Voix off ElevenLabs
Aujourd'hui
01
Corriger la clé ElevenLabs (erreur 401) EN COURS
La clé ElevenLabs dans .env retourne 401. Vérifier et remplacer par la clé valide depuis elevenlabs.io → Profile → API Key.
# Sur le VPS :
nano ~/.affiliate-video-system/.env
ELEVENLABS_API_KEY=ta_vraie_clé
02
Corriger le Voice ID dans pipeline.js EN COURS
Codex a mis l'ancien Voice ID. Remplacer par ton Voice ID personnel créé avec Voice Design.
// Voice ID correct :
const VOICE_ID = "FIba8m5J0VX1qo9cMnRE";
Dans pipeline.js, remplace le voice_id par FIba8m5J0VX1qo9cMnRE partout. Puis copie sur VPS et relance.
03
Merge audio + vidéo avec FFmpeg À VALIDER
FFmpeg fusionne voiceover.mp3 + video.mp4 → final.mp4 avec voix off synchronisée.
ffmpeg -i out/video.mp4 -i voiceover.mp3 \
  -c:v copy -c:a aac -shortest \
  out/final.mp4
ElevenLabs API eleven_multilingual_v2 FFmpeg merge

Phase 05 — Prochain
Vidéos stock Pixabay
Jour 2
01
Clé API Pixabay gratuite
Créer un compte sur pixabay.com/api/docs → récupérer la clé API gratuite → ajouter dans .env.
pixabay.com/api/docs 100% gratuit
02
Téléchargement automatique des clips
Claude génère les mots-clés pour chaque scène. Le pipeline cherche et télécharge 3 clips Pixabay correspondants automatiquement.
// Scène 1 → "person working computer stressed"
// Scène 2 → "content creator editing video"
GET https://pixabay.com/api/videos/?q={keyword}&key={API_KEY}
Ajoute dans pipeline.js une fonction downloadPixabayClips(keywords) qui appelle l'API Pixabay, récupère la meilleure vidéo pour chaque mot-clé et la télécharge dans output/<tool>/assets/.
03
Intégration dans Remotion comme background
Les clips Pixabay deviennent le fond de chaque scène. Ratio : 40% stock / 60% démo. Textes animés par-dessus avec Remotion.
OffthreadVideo Remotion 40% stock / 60% démo Overlay textes
// Dans Remotion :
import {OffthreadVideo} from 'remotion';
<OffthreadVideo src={pixabayClip} />
<TextOverlay>{hookText}</TextOverlay>
